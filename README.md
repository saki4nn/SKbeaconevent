# 💎 SKbeaconevent — v1.0
**Підтримувана версія гри:** `1.16.5+`

---

### 📖 Про плагін
**SKbeaconevent** — це захоплюючий ігровий івент для вашого сервера, який вносить елементи змагання та хаосу. У визначений час у світі з'являється містичний Маяк. Гравці повинні об'єднатися або битися між собою, щоб зруйнувати його та отримати винагороду.

Маяк має власний запас міцності (HP) та захищається, накладаючи на нападників випадкові ефекти зілля. Це ідеальний варіант для урізноманітнення геймплею на Survival, ClanWar або Anarchy серверах.

---

### ✨ Ключові особливості

* **⏳ Автоматичний запуск**
  Вбудований таймер дозволяє івенту запускатися самостійно через налаштований проміжок часу (за замовчуванням — щогодини).
* **💔 Механіка міцності (HP)**
  Маяк не можна зламати одним кліком. Він має певну кількість життів, кожне з яких потрібно "відбити". При кожному ударі весь сервер бачить повідомлення про прогрес.
* **🔮 Магічний захист**
  З певним шансом гравець, що вдарив маяк, може отримати як позитивні, так і негативні ефекти (наприклад, Сліпоту або Силу), що додає елемент несподіванки.
* **📊 Динамічна голограма**
  Над маяком автоматично з'являється інтерактивний текст, який у реальному часі відображає залишок HP.
* **✨ Візуальні ефекти**
  Постійні частинки (particles) навколо маяка дозволяють гравцям легко знайти його здалеку та створюють атмосферу магічної події.

### 💻 Командний центр

| Команда | Опис | Доступ (Permission) |
| :--- | :--- | :--- |
| `/beacon delay` | Перевірити час до наступної появи маяка | `beacon.player` |
| `/beacon setspawn` | Встановити точку появи маяка (для адмінів) | `beacon.admin` |
| `/beacon start` | Примусово запустити івент зараз | `beacon.admin` |
| `/beacon stop` | Зупинити активний івент | `beacon.admin` |
